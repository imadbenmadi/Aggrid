Available Themes

Light Theme

The default and most commonly used theme. Clean white background with dark text. Best for office environments and general use.

CSS Class: ag-theme-light

Dark Theme

Dark background with light text. Easy on the eyes for low-light environments. Modern appearance with high contrast.

CSS Class: ag-theme-dark

Balham Theme

Professional and clean design with subtle borders and spacing. Great for business applications and dashboards.

CSS Class: ag-theme-balham

Balham Dark Theme

Dark variant of the Balham theme. Maintains the professional look while providing a dark appearance.

CSS Class: ag-theme-balham-dark

Material Theme

Based on Google Material Design principles. Modern, flat design with material colors and interactions.

CSS Class: ag-theme-material

Applying Themes

Apply a theme by adding the CSS class to the grid container.

HTML: 
<ag-grid-angular
  class="ag-theme-light"
  [rowData]="rowData"
  [columnDefs]="columnDefs"
>
</ag-grid-angular>

Switching Themes Dynamically

Use property binding to change themes at runtime.

HTML: 
<ag-grid-angular
  [ngClass]="currentTheme"
  [rowData]="rowData"
  [columnDefs]="columnDefs"
>
</ag-grid-angular>
TypeScript: 
currentTheme: string = 'ag-theme-light';

switchTheme(newTheme: string) {{
  this.currentTheme = newTheme;
}}

Customizing Themes

Override theme variables with SCSS/CSS variables. Each theme exposes customizable variables for colors and sizing.

SCSS Override Example: 
::ng-deep .ag-theme-light {{
  --ag-background-color: #f0f0f0;
  --ag-border-color: #ddd;
  --ag-cell-horizontal-padding: 12px;
  --ag-row-height: 35px;
}}

Theme-Specific Features

Each theme provides different styling for:

  • Grid and cell backgrounds
  • Text colors and fonts
  • Header styling
  • Borders and dividers
  • Hover and selection states
  • Icons and controls

Best Practices

✓ Choose theme early: Decide on a theme at project start for consistency.
✓ Respect user preferences: Allow users to switch between light/dark themes.
✓ Test accessibility: Ensure sufficient contrast in chosen theme.
✓ Use CSS variables: Override theme colors consistently across your app.
✓ Load theme CSS: Include AG Grid theme CSS in your main styles.
