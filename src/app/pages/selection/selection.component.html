Selection Types

Checkbox Selection

Add a column with checkboxSelection: true to enable checkboxes. Set headerCheckboxSelection: true to include a select-all checkbox.

Example: 
{{
headerName: 'Select',
field: 'select',
checkboxSelection: true,
headerCheckboxSelection: true,
width: 80
}}

Getting Selected Rows

Use getSelectedRows() to retrieve all selected row data. Also use the selectionChanged event to react when selection changes.

TypeScript: 
onSelectionChanged() {{
  this.selectedRows = this.gridApi?.getSelectedRows();
  console.log(this.selectedRows);
}}

Selecting/Deselecting Programmatically

Use the Grid API methods to control selection programmatically.

TypeScript: 
selectAll() {{
  this.gridApi?.selectAll();
}}

deselectAll() {{
  this.gridApi?.deselectAll();
}}

Row Click Selection

Set rowSelection: 'single' or 'multiple' to enable selection by clicking the entire row.

HTML: 
<ag-grid-angular
  rowSelection="multiple"
  (selectionChanged)="onSelectionChanged()"
  ...
>
</ag-grid-angular>

Best Practices

✓ Use checkboxes for clarity: Checkboxes make selection intent obvious.
✓ Show selection count: Display how many rows are selected.
✓ Provide bulk actions: Enable actions on selected rows (delete, export, etc.).
✓ Show summary data: Calculate totals or summaries from selected rows.
✓ Confirm destructive actions: Ask for confirmation before deleting selected rows.
