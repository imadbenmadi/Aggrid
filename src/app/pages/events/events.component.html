Common Grid Events

Grid Lifecycle Events

  • gridReady: Fired when grid is initialized
  • firstDataRendered: Fired when first set of rows rendered
Example: 
(gridReady)="onGridReady($event)"
(firstDataRendered)="onFirstDataRendered($event)"

Row Events

  • rowClicked: When a row is clicked
  • rowDoubleClicked: When a row is double-clicked
  • selectionChanged: When selection changes
TypeScript: 
onRowClicked(event: any) {{
  console.log('Clicked row:', event.data);
  console.log('Clicked column:', event.colDef?.headerName);
}}

Cell Events

  • cellClicked: When a cell is clicked
  • cellDoubleClicked: When a cell is double-clicked
  • cellValueChanged: When a cell value is edited
  • cellEditingStarted: When editing starts
  • cellEditingStopped: When editing ends
TypeScript: 
onCellClicked(event: any) {{
  console.log('Cell value:', event.value);
  console.log('Column:', event.colDef?.field);
  console.log('Row index:', event.rowIndex);
}}

Data Events

  • sortChanged: When sorting changes
  • filterChanged: When filtering changes
  • paginationChanged: When pagination changes
TypeScript: 
onSortChanged(event: any) {{
  const state = this.gridColumnApi?.getColumnState();
  console.log('Current sort:', state?.filter(col => col.sort));
}}

Getting Event Parameters

Each event provides detailed information about what triggered it. Common properties: data, colDef, rowIndex, value, api, columnApi.

TypeScript: 
onRowClicked(event: any) {{
  event.data;           // Row object
  event.colDef;         // Column definition
  event.rowIndex;       // Row number
  event.api;            // Grid API
  event.columnApi;      // Column API
}}

Best Practices

✓ Use gridReady for initialization: Always initialize grid state in gridReady handler.
✓ Cache grid references: Store gridApi and columnApi for later use.
✓ Handle events efficiently: Avoid heavy computations in event handlers.
✓ Update UI based on events: React to user interactions to show/hide controls.
✓ Log important events: Maintain audit trails for data modifications.
