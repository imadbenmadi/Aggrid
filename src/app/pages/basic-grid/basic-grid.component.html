Basic Grid Setup

Learn how to set up your first AG Grid with data

AG Grid Basic Example
This is a simple AG Grid displaying employee data with basic column definitions.

What's Happening Here?

Column Definitions

The columnDefs array defines what columns should appear in the grid. Each object in the array specifies a column with properties like headerName, field, and width.

TypeScript Code: 
columnDefs = [
  {{ headerName: 'Name', field: 'name', width: 150 }},
  {{ headerName: 'Age', field: 'age', width: 100 }},
  {{ headerName: 'Country', field: 'country', width: 150 }}
];

Row Data

The rowData array contains your actual data. Each object in the array represents one row. The field property in columnDefs tells AG Grid which property from the data object to display in that column.

TypeScript Code: 
rowData = [
  {{ name: 'Alice', age: 28, country: 'USA' }},
  {{ name: 'Bob', age: 35, country: 'Canada' }},
  // ...
];

HTML Template

In the template, use the <ag-grid-angular> component. The [rowData] and [columnDefs] are property bindings. The (gridReady) event fires when the grid is initialized, allowing you to access the grid API for programmatic control.

HTML Template: 
<ag-grid-angular
  class="ag-theme-light grid-container"
  [rowData]="rowData"
  [columnDefs]="columnDefs"
  (gridReady)="onGridReady($event)">
</ag-grid-angular>

Grid API

When gridReady fires, you receive the grid API and column API. These objects provide methods to control the grid programmatically, such as sorting, filtering, updating data, and more.

TypeScript Code: 
onGridReady(params: any) {{
  this.gridApi = params.api;
  this.gridColumnApi = params.columnApi;
  // Now you can use gridApi methods
}}

Best Practices

✓ Always set container height: Use CSS to set a fixed or percentage height on the grid container.
✓ Use unique field names: Ensure your data object properties match the field names in columnDefs.
✓ Define column widths: Specify column widths to ensure consistent layout.
✓ Theme class required: Always add a theme class like ag-theme-light to the grid.
✓ Store grid reference: Keep references to gridApi and gridColumnApi for later use.
Key Concepts
Column Definition

Object that defines how a column should look and behave

Row Data

Array of objects representing the rows to display

Grid API

Methods to programmatically control the grid

Column API

Methods to manage columns and their properties

Grid Ready Event

Fired when grid is initialized and ready to use

Common Properties
  • headerName - Column header text
  • field - Data property name
  • width - Column width in pixels
  • hide - Hide/show column
  • sortable - Enable sorting
  • filter - Enable filtering
  • resizable - Allow column resize
